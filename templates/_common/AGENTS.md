# Common Template Files

Files shared across all project templates:

| File | Purpose |
|------|---------|
| `_gitignore` | Standard .gitignore for tech stack |
| `_editorconfig` | EditorConfig settings |
| `_env.example` | Environment variable template |
| `_dockerignore` | Docker ignore patterns |
| `Makefile` | Common task runner commands |
| `README.md` | Project README template |
| `CHANGELOG.md` | Keep a Changelog format |
| `CONTRIBUTING.md` | Contribution guidelines |
