# Recommended Libraries & Tools

## Cross-Platform

| Tool | Purpose | When to Use |
|------|---------|-------------|
| **pnpm** | Package manager | JS/TS projects (faster than npm/yarn) |
| **Docker** | Containerization | All projects (standardized environments) |
| **Make** | Task runner | Cross-project command consistency |
| **CommitLint** | Commit convention enforcement | All git projects |
| **Husky** | Git hooks | JS/TS projects |
| **EditorConfig** | Editor consistency | All projects |
| **Prettier** | Code formatter | JS/TS/HTML/CSS |
| **Renovate** | Dependency updates | All projects (auto-PR deps) |
| **Snyk / Trivy** | Security scanning | All projects |
| **Semantic Release** | Automated versioning | Libraries |
