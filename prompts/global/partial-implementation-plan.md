# Implementation Plan Prompt

Given a task specification, produce an implementation plan:

1. **Files to modify** (list with paths)
2. **Changes per file** (what to add/remove/change)
3. **Dependencies** (new packages, services, configs)
4. **Testing strategy** (what tests to write/update)
5. **Migration** (DB migrations, config migration, data migration)

If the task takes > 30 minutes of AI work, propose a phased approach.
