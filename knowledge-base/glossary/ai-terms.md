# AI Tooling Glossary

| Term | Definition |
|------|------------|
| **MCP** | Model Context Protocol — standard for AI tools to interact with external systems |
| **Agent** | AI-powered autonomous task executor with tool access |
| **Skill** | Reusable instruction set for specific AI tasks |
| **Tool** | Capability exposed to AI (read file, run command, search web) |
| **System Prompt** | Base instructions defining AI behavior |
| **Context Window** | Maximum tokens an AI model can process at once |
| **RAG** | Retrieval-Augmented Generation — fetching external data for context |
| **Fine-tuning** | Training a model on domain-specific data |
| **Vector Embedding** | Numeric representation of text for semantic search |
| **Spec-Driven Dev** | Writing specifications before implementation code |
